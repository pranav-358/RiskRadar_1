## 🎬 ANIMATIONS

### On Page Load
1. Logo slides in from top (0.8s)
2. Header fades in (0.8s, 0.2s delay)
3. Form fields slide up (staggered 0.3-0.5s)
4. Buttons slide up (0.7s delay)
5. Promo section slides up (0.3-0.6s delay)

### On Hover
- Buttons: Move up + shadow
- Links: Underline animation
- Form inputs: Scale + shadow
- Social buttons: Border color change

### Continuous
- Background shapes: Float smoothly
- Button shimmer: Continuous effect

## 🔧 TROUBLESHOOTING

### Issue: Colors are wrong
**Solution**: Clear browser cache (Ctrl+Shift+Delete)

### Issue: Layout broken on mobile
**Solution**: Check viewport meta tag in base.html

### Issue: Animations not smooth
**Solution**: Check browser support, try different browser

### Issue: Form not submitting
**Solution**: Check form action, verify CSRF token

### Issue: Responsive not working
**Solution**: Check media queries, test with DevTools
